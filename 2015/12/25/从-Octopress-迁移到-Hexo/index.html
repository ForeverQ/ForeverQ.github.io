从 Octopress 迁移到 Hexo

发表于 2015-12-25   |   分类于 Blog

为什么要迁移呢？因为相中了一款主题。

逛 Hexo 官网的时候看到了这款主题 NexT，一见钟情，加上之前已对 Hexo 有所耳闻，果断迁移。

在这里要特别感谢主题的开发者 iissnan。

关于 Hexo，其 官方文档（中文版）已经讲述的十分详尽，这里只做简单叙述。

环境部署

直接把 Hexo 装在手头上的 Windows 似乎不是个好选择，好在有虚拟机管理工具 Vagrant。我会在 另一篇 详细介绍一下 Vagrant，这里不再赘述。安装好 Ubuntu 镜像后就开始安装 Hexo。

官方文档介绍说 Hexo 依赖以下两个应用程序：

  • Node.js
  • Git

Git 的安装没什么阻碍，Ubuntu 环境下一条 sudo apt-get install git 就可以搞定。但是 Node.js 就会相对麻烦些。官方推荐的方法是先装 nvm，通过 cURL：
